Cursor Control

Allows you to hide or change the OS X cursor when it's over the viewer window.

Cursor Control

  • Enable
  • Style
  • Hide
Enable

Enables the cursor control patch

Style

The style of the cursor to display

Hide

Enable this port to hide the cursor when its over the viewer window.
