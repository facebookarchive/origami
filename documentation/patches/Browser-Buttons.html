Browser Buttons

Gives you the state of the back and forward buttons from the Browser Chrome patch. Useful for prototyping navigating between pages on the desktop web.

Browser Buttons

  • Back
  • Forward
Back

A boolean that changes between 0 and 1 based on if the back button is down or not.

Forward

A boolean that changes between 0 and 1 based on if the forward button is down or not.
