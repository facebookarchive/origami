Bouncy Animation

This patch lets you animate a value using a spring animation. The Pop Animation patch is the more modern version of this patch with easier to use controls and Code Export support, so it’s recommended that you use it instead. However, if you’re trying to animate something in an Iterator, you'll usually want to use this patch instead of Pop Animation since this will be able to animate each iteration independently.

Bouncy Animation

  • Number
  • Friction
  • Tension
  • Progress
Number

The number to animate. Usually a 0 or 1 value from a Switch or an Interaction patch.

Friction

The friction of the spring animation.

Tension

The tension of the spring animation.

Progress

The current value of the animation. Usually between or around 0 and 1.
