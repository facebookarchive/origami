Desktop Scroll

Freely scroll a Layer using a trackpad or a Magic Mouse.

To use, connect the image you want to scroll to both the Desktop Scroll and Layer patches, and connect the Desktop Scroll's X and/or Y positions to the Layer. The Layer needs to be set to an Anchor Point of Center.

Desktop Scroll

  • Enable
  • Image
  • Direction Locking
  • X Position
  • Y Position
Enable

When turned off, the patch will ignore scroll events. Any scroll animations previously started will continue.

Image

The image you want to scroll. The patch uses this to determine the size of what you're scrolling. This image should be the same size as the layer that you intend to scroll.

Direction Locking

Locks each scroll gesture to its starting direction (vertical or horizontal). Disable this to allow for diagonal scrolling.

X Position

The horizontal position of the scroller. Connect to the X Position of the Layer you want to scroll horizontally.

Y Position

The vertical position of the scroller. Connect to the Y Position of the Layer you want to scroll vertically.
