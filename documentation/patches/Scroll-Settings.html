Scroll Settings

Sets basic settings of a Scroll patch, such as resetting the scroll position, content size, bounds size, page padding, and direction locking.

Content Width

The width of the content being scrolled. Typically this is automatically detected when you plug an Image into the Scroll patch.

Content Height

The height of the content being scrolled. Typically this is automatically detected when you plug an Image into the Scroll patch.

Bounds Width

The width of the scroll view. Typically this is automatically detected from the Layer Group the Scroll patch is inside. Overriding this can be useful when you want an image to be scrolled in a smaller area than what is visible.

Bounds Height

The height of the scroll view. Typically this is automatically detected from the Layer Group the Scroll patch is inside. Overriding this can be useful when you want an image to be scrolled in a smaller area than what is visible.

Direction Locking

Locks each scroll gesture to its starting direction (vertical or horizontal). Disable this to allow for diagonal scrolling.

Jump Style

Lets you specify whether you want jumping to be immediate or animated when you pulse the Jump ports.

Jump X Position

A horizontal position between 0 and 1 that you want to scroll to when you pulse the Jump port.

Jump Y Position

A vertical position between 0 and 1 that you want to scroll to when you pulse the Jump port.

Jump to X

Pulse this to scroll the content to the specified Jump X Position.

Jump to Y

Pulse this to scroll the content to the specified Jump Y Position.

Page Padding

Adds padding between pages when using Paging or Wheel of Fortune scrolling.

Advanced Settings

Connect to a Scroll Advanced Settings patch for additional control.

Settings

Connect to the Settings port on the Scroll patch.
