Scroll Advanced Settings

Lets you configure advanced settings for a Scroll patch. The output can be plugged into the Settings port on the Scroll patch or the Scroll Settings patch.

Page Padding X

Allows you to specify padding in the horizontal direction when using Paging or Wheel of Fortune scrolling. Overrides Page Padding in Scroll Settings.

Page Padding Y

Allows you to specify padding in the vertical direction when using Paging or Wheel of Fortune scrolling. Overrides Page Padding in Scroll Settings.

Scale

Scales the position of the scroller (not the representation of the image on screen). Useful in cases where the object you're scrolling changes scale and you want it to be scrollable in both scales. E.g. the feed strip in Facebook Paper is scrollable when it's scaled down at the bottom of the screen and when it's scaled to full screen.

Scrollbars

Calculate scroll bar information that can be accessed with the Scroll Info patch.

Scrollbar Color

Choose between a Dark or Light scrollbar. This will show up in the Scrollbar output from the Scroll Info patch.

Scrolling Friction

A value between 0 and 1 that controls how quickly the scroller decelerates in Free Scrolling and Wheel of Fortune modes.

Rubberband Tension

A value between 0 and 1 that affects how the scroller feels when you drag past the edge of the scroller and release your finger.

Rubberband Friction

A value between 0 and 1 that affects how the scroller feels when you drag past the edge of the scroller and release your finger.

Rubberband Slip

A value between 0 and 1 that affects how much of your finger movement is applied to the scroller once you pass the edges.

Custom Inputs

Enables custom X/Y Inputs below.

Sample Inputs

Whether the scroll patch should be overridden with the custom inputs. This is usually connected to a boolean value that reflects whether a touch is down on the scroller.

X Input

Takes a value in units (not pixels) and uses the amount it changes to override the standard click/tap scroll input.

Y Input

Takes a value in units (not pixels) and uses the amount it changes to override the standard click/tap scroll input.

X Force

Insert a horizontal velocity in units (not pixels) into the scroller.

Y Force

Insert a vertical velocity in units (not pixels) into the scroller.

Settings

Connect to the Settings port on the Scroll patch or the Scroll Settings patch.
