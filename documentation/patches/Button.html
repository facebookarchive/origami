Button

Shows a button with a customizable label and background. Connect it to the Interaction 2 patch to make it interactive.

Enable

Enable or disable the button.

Anchor Point

A point in the containing Layer Group this layer is positioned relative to.

X Position

The horizontal position in pixels.

Y Position

The vertical position in pixels.

Width

The width in pixels.

Height

The height in pixels.

Background Image

The image to show behind the button title.

Background Color

The fill color of the button background.

Title

The button label.

Title Color

The color of the label.

Title Font Name

The font to use for the label.

Title Font Size

The size of the font in pixels.

Title Offset

The vertical position of the label relative to the background.

Corner Radius

The radius of the corners in pixels.

Opacity

The opacity of the layer. From 0% (0) to 100% (1).

Scale

The scale of the layer. 100% (1) is the default scale and 0% (0) is completely hidden.
