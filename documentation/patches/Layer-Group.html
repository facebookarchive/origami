Layer Group

A layer group lets you group multiple layers and have them show up on a single layer. It’s useful for organizing your projects as well making it easier to animate multiple layers at the same time. It’s similar to the Layer Groups in Sketch and Photoshop, except that it has a size that it will crop its contents to.

Layer Group

  • Render
  • Texture Target
  • Texture Mipmapping
  • Pixels Wide
  • Pixels High
  • Image
Render

Turning this off will disable all the patches inside the Layer Group from running. Using this port to turn off rendering when your layer group isn’t visible may help you improve performance. However, note that Origami Live does not support turning this port off at this time (although it will support disabling Layers to achieve similar performance gains).

Texture Target

Quartz Composer legacy option. Ignore this.

Texture Mipmapping

Quartz Composer legacy option. Ignore this.

Pixels Wide

The width of the Layer Group in pixels.

Pixels High

The height of the Layer Group in pixels.

Image

The image of the layer group. Must be connected to a Layer to have it show up.
