Border

Draws a border of a given thickness and color around the edges of the layer group it’s inside of. The border is similar to an inner stroke and has rectangular corners.

Border

  • Enable
  • Thickness
  • Color
  • Alpha
Enable

Turns the border on and off.

Thickness

How thick the border is in pixels on each edge.

Color

The color of the border.

Alpha

Adjusts the opacity of the border.
