Text Layer

Lets you show and style text.

Text Layer

  • Enable
  • Anchor Point
  • Text
  • Font Name
  • X Position
  • Y Position
  • Width
  • Height
  • Shadow Info
  • Color
  • Opacity
  • Scale
Enable

When false, the Layer is invisible

Anchor Point

Like the Layer patch, the Anchor Point changes how the X/Y Position behave in relation to the containing Layer Group. For Text Layers, the Anchor Point also changes the text alignment.

Text

The text you want to display, which can contain line breaks. You can dynamically change text by using patches like String Printer or Multiplexer (set to String type). Colored Emoji are not supported.

Font Name

The font family with which to render the text. For example HelveticaNeue-Bold

X Position

The horizontal position in pixels.

Y Position

The vertical position in pixels.

Width

The width in pixels of the text box. If the width is 0, it will auto-size to a single line.

Height

The height in pixels of the text box. If the height is 0, it will auto-size to the required amount of lines.

Shadow Info

Connect a Shadow Info patch to generate a text shadow.

Color

The color of the text.

Opacity

How visible the text is from 0 (hidden) to 1 (visible).

Scale

The scale of the text.
